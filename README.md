<h2><b>쇼핑몰 프로젝트</b></h2>

mvc 패턴을 도입한 쇼핑몰의 기본적인 기능과 구현 <br>
쇼핑몰을 이용하는 회원의 상품 구매 <br>
쇼핑몰을 관리하는 관리자의 쇼핑몰 운영<br>

<h2><b>사용 기술</b></h2>
<ul>
  <li>Spring boot</li>
